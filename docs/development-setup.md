# Development Setup
[![pt-br](https://img.shields.io/badge/lang-pt--br-green.svg)](https://github.com/pedrocapinos/FormLibrary/blob/main/docs/development-setup.pt-BR.md)

## Prerequisites

- Node.js 18+
- Angular CLI 17+

## Code Formatting

The project uses [Prettier](https://prettier.io/) for consistent formatting. Configuration is in `.prettierrc`:

- Print width: 100
- Single quotes, trailing commas (ES5)
- 2-space indentation, no tabs

Run `npx prettier --check .` to verify formatting, or `npx prettier --write .` to auto-format.
