# Configuração de Desenvolvimento
[![en](https://img.shields.io/badge/lang-en-red.svg)](https://github.com/pedrocapinos/FormLibrary/blob/main/docs/development-setup.md)

## Pré-requisitos

- Node.js 18+
- Angular CLI 17+

## Formatação de Código

O projeto usa [Prettier](https://prettier.io/) para formatação consistente. A configuração está em `.prettierrc`:

- Largura de linha: 100
- Aspas simples, vírgulas finais (ES5)
- Indentação de 2 espaços, sem tabs

Execute `npx prettier --check .` para verificar a formatação, ou `npx prettier --write .` para formatar automaticamente.
